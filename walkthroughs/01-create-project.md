# Welcome to C# Toolbox

Get your first .NET project running in seconds with the **Create Project** wizard.

## What you'll do

1. Open the command palette: `Ctrl+Shift+P` (Windows / Linux) or `Cmd+Shift+P` (macOS).
2. Type **C# Toolbox: Create Project** and press `Enter`.
3. Pick a template (Console, Web API, Worker Service, Aspire, …) — there are recent-project chips to reuse the last few.
4. Name your project & solution, choose a folder.
5. Pick your target framework — **.NET 9** is recommended (look for the ★).
6. Click **Create Project** and watch your project come to life.

> **Shortcut:** `Ctrl+Alt+/` then `P` jumps straight to the wizard.

Once the project is created, the wizard auto-closes and your new solution appears under the chosen folder. Open it with **File → Open Folder…**.
